> **专利权人**：江苏瑞霆生物科技有限公司
> **申请日**：2021.11.30 | **授权公告日**：2022.12.13

## 一、核心配方组成（质量百分比）

| 组分        | 含量范围        | 说明               |
| --------- | ----------- | ---------------- |
| **植物油脂**  | **1 ~ 40%** | ⭐ **油占比低**，可低至1% |
| **氢化卵磷脂** | 8 ~ 20%     | 主乳化剂，形成液晶结构      |
| **糖酯**    | 1 ~ 10%     | 助乳化剂，增强渗透性       |
| **助溶剂**   | 10 ~ 50%    | 甘油、二乙二醇、聚乙二醇等    |
| **水**     | 余量至100%     | 连续相              |
|           |             |                  |
|           |             |                  |

---

## 二、⭐ 油占比低 — 关键亮点

### 🔴 核心优势：植物油脂仅需 **1% ~ 40%**

- **下限极低**：油相最低可仅 **1%**，远低于传统乳液的油相占比（通常40%以上）
- **高效载体**：即使低油含量，仍能实现高效包载和透皮递送
- **成本与肤感优化**：低油相带来更清爽肤感，同时降低油脂原料成本

> 专利强调：该配方通过**氢化卵磷脂 + 糖酯**的乳化体系，在低油含量下仍能形成**稳定的液晶纳米结构**，实现优异的缓释、促渗和保湿功能。

---

## 三、糖酯种类与结构控制（关键创新点）

### 可选糖酯
1. **鼠李糖脂** — 双-鼠李糖脂与单-鼠李糖脂复配
   - 质量比：**双-鼠李糖脂 : 单-鼠李糖脂 ≥ 3:1**（优选 3:1）
   - 双酯比例高 → 渗透性和防腐杀菌能力更优
2. **槐糖脂** — 体系 pH ≥ 7 时，为**内酯型结构**
   - 内酯型结构比酸型结构（pH=5）透皮性更佳，防腐杀菌能力更强
3. **甘露糖赤藓糖醇酯** — 进一步增强渗透性

## 五、制备方法

```
油相制备：植物油脂 → 加热搅拌
        ↓
水相制备：氢化卵磷脂 + 糖酯 + 助溶剂 + 水 → 水浴加热，搅拌溶解
        ↓
乳化：油相加入水相 → 高速分散（15000 rpm，3 min）
        ↓
均质：高压均质（80 MPa，循环6次）→ 液晶纳米乳液
```

## 七、主要功效

| 功效 | 机制 |
|------|------|
| **高渗透性** | 液晶纳米结构 + 糖酯促渗，有效成分高效进入表皮/真皮 |
| **保湿** | 液晶结构锁水，修复皮肤屏障 |
| **杀菌/防腐** | 糖酯（尤其鼠李糖脂双酯、内酯型槐糖脂）自带抗菌活性 |
| **消炎修复** | 植物油脂活性成分 + 糖酯协同作用 |
| **良好肤感** | 低油相 + 纳米级粒径，清爽不油腻 |

---

## 八、应用方向

适用于化妆品配方：洗面奶、面膜、化妆水、**乳液**、膏霜、精华等。

---

## 九、核心创新点总结

1. **低油相液晶纳米乳液**：油脂仅1%~40%，通过氢化卵磷脂+糖酯形成稳定液晶结构
2. **糖酯结构精确调控**：控制鼠李糖脂双/单酯比例、槐糖脂内酯/酸型比例，优化渗透与防腐性能
3. **高透皮吸收**：有效成分从皮肤表面进入表皮和真皮，在靶点部位积聚，生物利用度高
4. **多功能一体化**：促渗 + 保湿 + 杀菌 + 消炎 + 修复屏障
