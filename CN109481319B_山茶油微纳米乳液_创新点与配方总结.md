# 一种山茶油微纳米乳液及其制备方法 — 创新点与配方总结

> 专利号：CN 109481319B  
> 专利权人：杭州千岛湖天鑫有限公司  
> 发明人：董全喜、邓学阳

---

## 一、技术背景与待解决的问题

1. **山茶油本身粘稠**，日常皮肤护理直接使用不方便。  
2. **传统乳化法**所得护肤品感官效果差，液滴粒径大，在介质中分散性能差，皮肤不易吸收。  
3. **现有微乳液技术**（如 CN103283866B）存在明显缺陷：山茶油负载量过低（仅 0.1~4%），表面活性剂用量过大（5~70%），对皮肤刺激性强，使用风险高。  
4. **现有护肤油技术**（如 CN102885753B）油滴粒径大、分散性差，整体感官粘稠，皮肤不易吸收。

---

## 二、核心创新点

### 2.1 高负载天然山茶油微纳米体系
- **山茶油含量高达 35~50%**（优选 40~45%），远高于现有微乳液技术，显著提升了滋润保湿与护肤功效。  
- 通过微射流高压均质技术将液滴粒径控制在 **100 nm 左右**（实施例平均粒径 89~132 nm），实现透明/半透明外观，同时保证优异的皮肤渗透性和吸收性。

### 2.2 天然安全、皮肤高亲和性配方设计
- **磷脂（卵磷脂/氢化卵磷脂）** 作为乳化剂：磷脂酰胆碱含量 ≥75%，本身就是人体细胞膜重要组成成分，兼具乳化、修复皮肤屏障和滋润多重功能。  
- **甘油作为外相主体溶剂**（替代传统水基体系）：  
  - 甘油的折光率（~1.47）更接近山茶油（~1.45），更容易实现透明/半透明的光学效果；  
  - 甘油本身具有优异的保湿滋润性能。  
- **天然抗氧化剂**（维生素E 或 迷迭香提取物）：避免合成物质对皮肤的刺激，安全性高。

### 2.3 简洁高效的制备工艺
- 采用 **微射流高压均质机**（150~250 MPa，循环 2~5 次），工艺步骤简洁，可工业化放大。  
- 通过高速预混（5000~6000 rpm，5~10 min）+ 高压均质的组合，确保体系均匀稳定。

### 2.4 综合性能优势
经 60 例受试者肤感评价（外观、涂抹性、吸收速度、滋润保湿）验证，本发明各实施例总分（29~34 分）**显著优于市售山茶油乳液**（19~22 分），且在 45°C 加速储存 2 个月条件下保持稳定。

## 四、制备方法

### 4.1 工艺流程

```
A相制备 → B相制备 → 高速预混 → 微射流高压均质 → 降温出料
```

### 4.2 详细步骤

**步骤 1：A相制备**
- 将磷脂加入甘油中，搅拌均匀；
- 加入水，加热至 **60~65°C**，持续搅拌至磷脂完全溶解均匀；
- 作为 A 相待用。

**步骤 2：B相制备与预乳化**
- 将抗氧化剂与山茶油混合均匀，作为 B 相；
- 将 B 相加入 A 相中，以 **5000~6000 rpm** 高速搅拌 **5~10 min**，得到预处理液。

**步骤 3：高压均质**
- 将预处理液通过 **微射流高压均质机**；
- 设置均质压力：**150~250 MPa**；
- 循环次数：**2~5 次**；
- 出料后降温（静置或缓慢搅拌）至温度 **≤ 37°C**，即得山茶油微纳米乳液。

## 六、总结

本专利通过 **"高含量天然山茶油 + 高纯度磷脂乳化 + 甘油外相体系 + 微射流高压均质"** 的组合创新，成功解决了传统山茶油护肤品粘稠难用、粒径大、吸收差以及现有微乳液技术负载低、刺激性强等核心痛点。所得产品兼具 **高负载、高渗透、高稳定、高安全、高颜值（透明/半透明）** 五大优势，具有良好的化妆品开发应用前景。
