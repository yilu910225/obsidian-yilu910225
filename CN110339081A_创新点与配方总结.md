# CN110339081A 一种稳定的油溶多肽组合物及其制备方法和应用

## 一、创新点总结

### 1. 技术领域与问题
- **领域**：化妆品领域，特别涉及全油配方（无水相）化妆品中的多肽应用。
- **现有问题**：
  - 多肽产品通常为水溶性冻干粉或水溶液，无法与油性配方互溶。
  - 现有油性化妆品中添加的肽类物质极不稳定，对温度敏感，易变色、浑浊或分层。
  - 水溶性多肽在油性环境中易氧化、水解，且需添加防腐剂，存在安全隐患。

### 2. 核心创新方案
- **油溶多肽组合物**：通过**多肽 + 特定增溶剂 + 油性溶剂**的三元体系，实现多肽在油相中的稳定溶解。
- **关键增溶剂筛选标准**：
  - 辛醇水分配系数为 **1.70–1.90**，或
  - HLB值（亲水亲油平衡值）为 **8.0–10.0**。
- **优选多肽**：氨基酸数目 ≤5，选自棕榈酰二肽-7、棕榈酰三肽-1、棕榈酰三肽-5、棕榈酰三肽-8、棕榈酰四肽-5、棕榈酰四肽-7、棕榈酰四肽-10、棕榈酰五肽-4中的至少一种。
- **优选增溶剂**：乙基己基甘油、甘油辛酸酯、脂肪酸聚氧乙烯酯、失水山梨醇月桂酸酯、聚氧乙烯氧丙烯油酸酯。
- **优选油性溶剂**：辛酸癸酸甘油三酯、棕榈酸乙基己酯、棕榈酸异丙酯、聚二甲基硅氧烷、辛基聚甲基硅氧烷、异十六烷。
- **可选增稠剂**：蜂蜡、三山嵛精、巴西棕榈蜡、地蜡（用于膏体剂型）。

### 3. 剂型与配方范围（按重量份）
| 组分 | 重量份 |
|------|--------|
| 增溶剂 | 1–20 份 |
| 增稠剂 | 0–30 份 |
| 油性溶剂 | 50–90 份 |
| 多肽含量 | 10–10000 ppm |

- **液体剂型**：不含增稠剂，澄清透明无色至淡黄色油状液体。
- **膏体剂型**：含增稠剂，白色或类白色膏体，加热至70℃可变为液体。

### 4. 制备方法
**液体剂型**：
1. 多肽与增溶剂混合，充分搅拌；
2. 加入油性溶剂，继续搅拌混合，即得。

**膏体剂型**：
1. 多肽与增溶剂混合，充分搅拌；
2. 加入油性溶剂，充分搅拌并加热（70–85℃），备用；
3. 增稠剂加热后，加入步骤2混合物中，搅拌均匀，冷却，即得。

### 5. 技术优势与效果
1. **相容性**：易于添加至全油配方化妆品中，解决美容多肽在油相中的应用难题。
2. **剂型灵活**：液体和膏体两种剂型，制备工艺简单，易于终端产品选择。
3. **高温稳定性**：能够耐受化妆品加工过程中的高温（90℃下2小时，多肽含量变化率 <1.5%）。
4. **长期稳定性**：25℃下6个月，多肽含量变化率 <1%，性状无变化。
5. **安全性**：油性环境不易长菌，无需添加防腐剂，配方更温和、安全，避免多肽被溶解氧氧化和水解。

## 三、对比例（验证增溶剂关键性）

| 对比例 | 与实施例区别 | 结果 |
|--------|--------------|------|
| 对比例1–3 | 不含增溶剂 | 白色浑浊液体/膏体，多肽部分不溶 |
| 对比例4 | 增溶剂为司盘80（HLB=4.5） | 白色浑浊液体，多肽部分不溶 |
| 对比例5 | 增溶剂为辛甘醇（辛醇水分配系数=1.67） | 白色浑浊液体，多肽部分不溶 |

> **结论**：增溶剂的辛醇水分配系数（1.70–1.90）或HLB值（8.0–10.0）是实现油溶和稳定性的关键参数，偏离此范围即无法获得澄清稳定体系。

## 四、应用配方

### 应用例1：抗皱抗衰精华油（1kg）
| 原料名称 | 配方量 |
|----------|--------|
| 实施例1油溶多肽组合物 | 50 g |
| 迷迭香叶油 | 5 g |
| 茴香油 | 5 g |
| 霍霍巴籽油 | 50 g |
| 葡萄籽油 | 890 g |

**工艺**：葡萄籽油、霍霍巴籽油、迷迭香叶油、茴香油升温至55–60℃搅拌均匀，冷却至40℃以下后加入实施例1组合物，搅拌均匀，过滤。

**稳定性**：45℃和-15℃保持1个月，性状无明显变化。

### 应用例2：黑发亮发修护发油（1kg）

| 香精 | 1 g |
| 维生素E | 2 g |
| 芝麻籽油 | 50 g |

**稳定性**：45℃和-15℃保持1个月，性状无明显变化。

### 应用例3：丰唇滋润唇膏（1kg）

**A组**：
| 原料名称 | 含量（%） |
|----------|-----------|
| 白凡士林 | 40 |
| 单硬脂酸甘油酯 | 25 |
| 鲸蜡 | 7 |
| 轻质矿物油 | 25 |
| 羟苯丙酯 | 0.8 |

**B组**：
| 原料名称 | 含量（%） |
|----------|-----------|
| 生育酚乙酸酯 | 1.0 |
| 香精 | 0.2 |
| 实施例3油溶多肽组合物 | 1.0 |

**工艺**：A组加热至85℃溶解搅拌均匀，降温至70–75℃，加入B组，混合搅拌均匀，注入模具，冷却定型。

**稳定性**：45℃和-15℃保持72h，恢复室温后外观无明显变化，能正常使用。
